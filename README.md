# Portfólio Retro Moderno

Projeto pessoal de portfólio com visual retro moderno, feito para ser simples de adaptar e reutilizar.

## Tecnologias
- React
- Vite
- JavaScript
- HTML + CSS

## Como usar (PT-BR)
1. Instale as dependências:
   - `npm install`
2. Rode localmente:
   - `npm run dev`
3. Build para produção:
   - `npm run build`
4. Pré-visualizar o build:
   - `npm run preview`

### Onde editar conteúdo
- Texto e seções: `src/App.jsx`
- Estilos: `src/App.css`
- Artigos: `src/data/articles.js`
- CV: `public/Cv.pdf`
- Foto: `public/avatar.jpg`
